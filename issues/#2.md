### User Story

As a **calculator user**,
I want to **multiply two numbers together**,
so that **I can perform multiplication beyond basic addition**.

---

### Background

The `OperationFactory` currently returns a `NullOperation` placeholder for `MULT`.
The `x` button is visible in the UI but pressing it produces no result.

---

### Acceptance Criteria

- [ ] **Given** I enter `4`, press `x`, enter `3`, and press `=`,
      **then** the display shows `12`.
- [ ] **Given** I enter `0`, press `x`, enter `9`, and press `=`,
      **then** the display shows `0`.
- [ ] **Given** I enter an invalid expression,
      **then** the display shows `Error` (consistent with existing behaviour).
- [ ] `OperationFactory` returns a `MultiplyOperation` for `CalcAction.MULT`.
- [ ] Unit tests cover: standard product, multiplication by zero, and invalid input.
- [ ] No new PMD violations introduced.

---

### Implementation Hints

1. Create `src/main/java/at/jku/se/calculator/operators/MultiplyOperation.java`
   implementing `ICalculationOperation`, analogous to `AddOperation`.
2. Register it in `OperationFactory.getOperation()` for `CalcAction.MULT`.
3. Add unit tests in a new `TestMultiplyOperation` test class
   (see `TestAddOperation` as reference).
