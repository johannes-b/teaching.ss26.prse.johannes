### User Story

As a **calculator user**,
I want to **subtract one number from another**,
so that **I can perform basic arithmetic beyond addition**.

---

### Background

The `OperationFactory` currently returns a `NullOperation` placeholder for all
operations except addition. The `-` button is visible in the UI but pressing it
produces no result.

---

### Acceptance Criteria

- [ ] **Given** I enter `8`, press `-`, enter `3`, and press `=`,
      **then** the display shows `5`.
- [ ] **Given** I enter `3`, press `-`, enter `8`, and press `=`,
      **then** the display shows `-5`.
- [ ] **Given** I enter an invalid expression,
      **then** the display shows `Error` (consistent with existing behaviour).
- [ ] `OperationFactory` returns a `SubtractOperation` for `CalcAction.SUB`.
- [ ] Unit tests cover: positive result, negative result, and invalid input.
- [ ] No new PMD violations introduced.

---

### Implementation Hints

1. Create `src/main/java/at/jku/se/calculator/operators/SubtractOperation.java`
   implementing `ICalculationOperation`, analogous to `AddOperation`.
2. Register it in `OperationFactory.getOperation()` for `CalcAction.SUB`.
3. Add unit tests in a new `TestSubtractOperation` test class
   (see `TestAddOperation` as reference).
